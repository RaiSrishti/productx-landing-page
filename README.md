# Modern Product Landing Page

##  Project Overview

Modern Product Landing Page is a responsive React-based UI project designed to demonstrate strong component structure, clean layout hierarchy, reusable components, and interactive user experience.

This project was built as part of an internship task to showcase frontend development skills using React.

---

##  Features

- Hero section with headline and call-to-action button
- Reusable feature cards (3 cards minimum)
- Responsive layout (mobile + desktop)
- Hover effects on buttons and cards
- Clean and structured component architecture

---

##  Technologies Used

- React (Vite)
- JavaScript (ES6+)
- Functional Components
- CSS (Responsive Design)
- Git & GitHub

##  How to Run the Project

1. Clone the repository:

git clone <your-repo-link>

2. Navigate into project folder:

cd modern-product-landing-page


3. Install dependencies:

npm install


4. Start development server:

npm run dev


5. Open browser:

http://localhost:5173
